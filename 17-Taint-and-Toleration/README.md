# Taint & Tolerations

## Taint the nodes ( worker2 )  
```
kubectl taint nodes worker2 app=myapp:NoSchedule
kubectl describe nodes worker2 | grep -i taint
```


## Untaint the nodes
```
kubectl taint nodes worker2 myapp-
```
